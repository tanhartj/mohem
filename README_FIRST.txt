🎉 خوش آمدید!

این پکیج شامل تمام فایل‌های پروژه YouTube Viral Machine است.

═══════════════════════════════════════════════════════════════

📋 فایل‌های مهم:

1. QUICK_START.md           - راه‌اندازی سریع (5 دقیقه!)
2. DEPLOY_INSTRUCTIONS_FA.md - راهنمای کامل نصب (فارسی)
3. RENDER_DEPLOY.md         - نصب روی Render.com (Cloud)
4. REPORT-FA.md             - گزارش بررسی پروژه
5. README-FA.md             - مستندات کامل فارسی
6. README.md                - مستندات انگلیسی

═══════════════════════════════════════════════════════════════

🚀 شروع سریع:

   1. باز کردن فایل: QUICK_START.md
   2. نصب Redis: docker run -d -p 6379:6379 redis:7-alpine
   3. نصب پکیج‌ها: npm install
   4. راه‌اندازی: npm start
   5. مشاهده: http://localhost:5000

═══════════════════════════════════════════════════════════════

📦 محتویات پکیج:

✅ کد منبع کامل (بدون باگ)
✅ تمام ماژول‌ها و قابلیت‌ها
✅ راهنمای نصب محلی
✅ راهنمای نصب Render.com
✅ فایل‌های Docker
✅ تنظیمات نمونه (.env.example)
✅ مستندات فارسی کامل

═══════════════════════════════════════════════════════════════

⚠️ نکات مهم:

- Redis ضروری است (دستورالعمل در QUICK_START.md)
- API Keys اختیاری هستند (پروژه بدون آنها هم کار می‌کند)
- برای Render باید .env را تنظیم کنید

═══════════════════════════════════════════════════════════════

🆘 کمک نیاز دارید؟

- همه راهنماها به فارسی هستند
- برای هر مشکل، REPORT-FA.md را بخوانید
