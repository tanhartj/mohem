═══════════════════════════════════════════════════════════════════════════════
  🚀 شروع سریع
═══════════════════════════════════════════════════════════════════════════════

1. استخراج فایل:
   tar -xzf youtube-viral-machine-complete.tar.gz
   cd youtube-viral-machine-complete

2. نصب وابستگی‌ها:
   npm install

3. نصب Redis (ضروری!):
   # macOS:
   brew install redis && redis-server
   
   # Linux:
   sudo apt-get install redis-server
   
   # Docker:
   docker run -d -p 6379:6379 redis:7-alpine

4. تنظیم environment:
   cp .env.example .env
   # ویرایش .env و تنظیم API keys

5. راه‌اندازی دیتابیس:
   npm run init-db

6. اجرا:
   npm run dev          # ترمینال 1
   npm run worker       # ترمینال 2

7. مرورگر:
   http://localhost:5000

فایل .env:

VIDEOS_PER_DAY=15                  # تعداد ویدیوها در روز
BASE_INTERVAL_MINUTES=96           # فاصله بین ویدیوها (دقیقه)
SCHEDULE_JITTER_MINUTES=15         # تصادفی‌سازی ±15 دقیقه
MAX_CONCURRENT_JOBS=3              # تعداد ویدیوهای همزمان در حال پردازش
UPLOAD_MAX_RETRIES=5               # تعداد تلاش مجدد آپلود
RETRY_BACKOFF_MS=60000             # تأخیر بین تلاش‌ها (میلی‌ثانیه)

═══════════════════════════════════════════════════════════════════════════════
  🔧 عیب‌یابی
═══════════════════════════════════════════════════════════════════════════════

❌ Redis Connection Refused
   → redis-cli ping   (باید PONG برگرداند)
   → یا از Upstash/Redis Cloud استفاده کن

❌ FFmpeg Not Found
   → brew install ffmpeg  (macOS)
   → apt-get install ffmpeg  (Linux)

❌ Out of Memory
   → MAX_CONCURRENT_JOBS=1 در .env

❌ YouTube Upload Failed
   → بررسی Client ID/Secret
   → Redirect URI صحیح باشد
